# 瑞控RKF405飞控固件仓库
## 介绍
这是扬州瑞控汽车电子有限公司RKF405对BetaFlight以及INAV两种固件所用的软件仓库；  
**根据BetaFlight和INAV的开源协议**，将需要替换的驱动代码以及端口配置开源上传；  
![Logo](./images/正面照.jpg)  
本仓库更新速度会跟进两个开源软件的进度,用户可找到固件配置文件、代码修改方法以及已经编译好的固件并刷写进飞控板；
